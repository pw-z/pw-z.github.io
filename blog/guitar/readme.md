# 吉他？

### 有疑虑解决疑虑别盲目练琴#20230618

最近学习游泳与练琴的情况形成强烈对比。游泳方面，蝶仰蛙自四种泳姿，每种泳姿大体有哪些待学习的内容，分什么步骤练习，速度、心率等客观的衡量指标是怎样，都比较清晰；吉他方面，乐理有音程、音阶、和弦等等，需要练耳，需要将乐理与指板联系起来熟悉指板，右手有节奏要练习，说是还可以说出一大堆，但这些内容没有融合起来不成体系，如此缺乏对当前专项进度（此时此刻在练习的内容练到什么程度即可）与整体进度（还有多少内容需要学习）的掌控感进而感到有些迷失。

### 练琴的时间投入#20230709

读了几篇关于时间投入的文章：
* [学习吉他大概要多久才能达到你想要的水平？](https://hubguitar.com/articles/how-long-does-it-take-to-learn-guitar)  
* [如何最有效率的练习吉他？](https://hubguitar.com/zh_han/articles/practice-success)  
* [学习吉他，每天练习多久？](https://hubguitar.com/zh_han/articles/how-much-time-to-practice-daily)

**每天练琴时间不要太少**。假设达到专业水平需要10000小时的练习时长，一个人每天练习30分钟持续55年时长上足够但很可能无法达到专业水平，因为每次练习需要热身时间然后才可能突破上限，每天30分钟在你达到一定水平后无法帮助你突破上限。

**固定的连续的练琴时间**。碎片时间练琴效率低。旅行、出差之类的事情都不是借口，可以搞把旅行吉他。

> 你花了多少时间来练习吉他，是决定你总体水平的最大因素。第二点是练习的质量，其余的是资源，天赋和运气*。
