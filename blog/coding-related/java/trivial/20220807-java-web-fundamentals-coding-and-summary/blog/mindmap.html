生命周期
1、加载&实例化：默认情况下，当Servlet第一次被访问的时候由web容器（Tomcat）创建&实例化

（可以配置成容器启动时就创建）

2、初始化：实例化后，容器将调用init()方法进行初始化，完成配置加载等操作，该方法只执行一次
3、请求处理：每次请求servlet时，容器都会调用一次service方法进行处理
4、服务终止：容器关闭时会调用destroy方法销毁servlet实例
