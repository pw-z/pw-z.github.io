# 理解素描

2021年3月1号，我正式开始学习素描，目标是可以画小人儿，这份文档用来追踪这个目标的实现。


```
碎碎念
---
经过一年多的学习，最初的目标拆分成如下两个：
  1. 理解素描（这份文档是具体的进展）
  2. 完成一组自己满意的人像习作（足够理解再谈满意）
两项都没有明确的终止界限，这本来也是一件非理性的爱好，应该会终止在某个随缘的节点上。

这份文档会包含详尽的学习、练习记录及相关思考，希望完结时顺便可以有这样的参考价值：
`一个人需要花多久时间、通过什么方法、做哪些练习、解决哪些问题，最终能够达到???水平。`
不是概括式的总结。完完整整的记录。
```
